# Employee Scheduling System

## Overview
A Java-based system for managing employee shift scheduling based on availability constraints.  
The system supports role-based functionality, allowing employees to submit availability and managers to assign shifts accordingly.

## Features
- Employee availability submission (constraints-based)
- Manager-controlled shift assignment
- Role separation between employees and managers
- Handling of conflicting constraints and schedule validation

## Technologies
- Java
- Object-Oriented Programming (OOP)
